Install on Linux:

```
git clone https://github.com/MrQbasic/SDR-SA
cd SDR-SA
git submodule init
git submodule update
mkdir build && cd build
cmake ..
make -j 4
./src/SDRSA
```


HOW DO I USE IT ?

1. init the source by clicking on it
2. enter Graph menu
3. add Graph
4. select source from list
5. doubble right-click on the spectrum to see it
